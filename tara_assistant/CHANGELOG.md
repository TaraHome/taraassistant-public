# Changelog

## 1.1.0

- AI-powered device onboarding agent for setting up new devices
- Web search integration (Brave Search / DuckDuckGo fallback) for device documentation
- Add-on management capabilities via Supervisor API
- Background device scanner with configurable interval
- Added `hassio_api` permission for Supervisor API access

## 1.0.0

- Initial release as Home Assistant add-on
- AI-powered natural language device control
- Automatic pattern detection and automation suggestions
- Support for OpenAI, Anthropic, Google, Ollama, and OpenAI-compatible providers
- Setup wizard with encrypted credential storage
- Safety guardrails for sensitive actions
